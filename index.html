Apa itu html

HTML itu adalah sebuah bahasa yang mengunakan markup atau penanda untuk membuat halaman web. penanda atau markup ini, nanti akan kita sebuat dengan tag. penanda atau markup ini, nanti akan kita sebut dengan html berperan untuk menetukan struktur konten dan tampilan dari sebuah web. Kalau kita ibarat kan nih, HTML itu seperti batu bara untuk membangun rumah. Batu bata ini dapat i susun, hiangga menjadi fondasi dasar, HTML memiliki kelebiahn seperti memiliki komunitas yang cukup besar dengan pengguna yang cukup luas. Tidak hanya itu, kelebiahan lainya adalah HTML mudah di pelajari oleh seseorang web developor pemula. Kelabihan lainnya adalah html merupakn bahasan pemprograman yang gratis atau open source.
